# Module 00: Solidifying the Bedrock

This module is a diagnostic and reinforcement module. Its goal is to verify and fortify your understanding of C fundamentals that are critical for tackling the advanced topics in later modules.

## Concepts to Master

1.  **Pointer Mastery:** Pointer arithmetic, function pointers, `void*` for generic programming.
2.  **Memory Layout:** `sizeof`, struct padding and alignment, and the difference between stack, heap, and static/global data segments.
3.  **Compilation Pipeline:** The stages of preprocessing, compilation, assembly, and linking. You can investigate this with `gcc -v my_program.c`.

## Pre-flight Check & Exercises

Before proceeding, you must be able to complete the following exercises. They are not just about getting the right answer, but understanding *why* the answer is right.

*   `exercises/ex1_pointers.c`: A series of challenges involving pointer arithmetic and function pointers.
*   `exercises/ex2_memory_layout.c`: Questions related to `sizeof` and struct padding that you answer by writing and running code.
